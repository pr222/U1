HTML5

Element och taggar

Ett element är helheten utav det som finns i och mellan en öppnande och stängande tagg. Vilket element som används beror på vilken tagg man använder, taggen definierar alltså vad elementet ska bestå av och den skrivs alltid inom pilar < >. För en rubrik skriver man då <h1> i html-dokumentet, sedan vad man vill att rubriken ska säga, för att sedan stänga rubrik-elementet med </h1>.

Attribut

Elementen kan tilldelas attribut i den öppnande namntaggen för att ändra vissa egenskaper eller förtydliga information. En h1-rubrik skulle exempelvis kunna få attributet id eller class (<h1 id="uniktId"> eller <h1 class="unikClass">) som sedan kan manipuleras med exempelvis CSS.
